## 1000: A+B Problem

### 题目描述

计算2个整数a, b的和.这两个整数都在1到1000之间.

### 输入

输入只有一行，包括2个整数a, b.两者之间用一个空格分开.

### 输出

输出只有一行，包括1个整数.必须用print()输出你的结果，才能通过OJ的评判.

### 代码

```python
#!/usr/bin/python3
a, b = map(int, input().split())
print(a+b)
```
